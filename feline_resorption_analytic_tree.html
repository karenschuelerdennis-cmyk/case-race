Tooth Resorption in Cats — Analytic Question Tree

Decision logic + methodology for the 2026–27 feline classification study (Peralta-style design)

Core aims: (1) Is the Andreasen human system applicable to felines? (2) If so, is it better than the AVDC feline system? (3) Can a superior universal feline system be built?

Premise / dataset Analytic question Methodology & metrics Decision node Outcome / conclusion
Background
State of classification before this study
Human dentistry: 17 resorption systems; Andreasen (2007) most widely accepted. Feline (vet): one system — AVDC (DuPont & DeBowes, 2000s). Canine (vet): Peralta (2010) compared Andreasen vs AVDC and found the human Andreasen system more applicable to dogs. This study asks whether the same holds for cats — and whether anything better can be built.
Step 0 · Shared dataset
Assemble a single feline resorption sample classified under both systems
  • Curate feline teeth with confirmed resorptive lesions (intraoral radiographs and/or CBCT; histology where available as ground truth).
  • Multiple calibrated, blinded observers independently classify each lesion under both Andreasen and AVDC.
  • Pre-register thresholds (e.g., "applicable" = near-100% coverage + ≥ substantial agreement, weighted κ ≥ 0.61).
1

Is the Andreasen (human) system applicable to felines?

Methodology · Q1
Apply Andreasen to feline lesions and measure fit (mirrors Peralta's canine step)
  • Coverage / completeness: proportion of feline lesions assignable to an Andreasen category (count "unclassifiable" cases).
  • Inter-observer reliability: Fleiss'/weighted Cohen's κ across observers.
  • Intra-observer reliability: repeat reads, weighted κ.
  • Construct validity: agreement of Andreasen categories with histologic/radiographic ground truth.
Decision · Q1
Does Andreasen meet the pre-set applicability thresholds in cats?
Coverage near-complete and reliability/validity acceptable?
▼ YES — applicable
Finding 1a
Andreasen is applicable to felines
Consistent with Peralta's canine result. Proceed to Phase 2 (head-to-head vs AVDC).
▼ NO — not applicable
Finding 1b
Andreasen does NOT transfer to felines
Diverges from the canine finding. AVDC remains the default; skip the head-to-head and go straight to Phase 3 (can a better feline-specific system be built?).
2

Is Andreasen better than the AVDC feline system?

Methodology · Q2 (Peralta comparison)
Head-to-head comparison on the same feline dataset
  • Reliability: compare inter-/intra-observer κ for Andreasen vs AVDC.
  • Validity: compare each system's correlation with ground-truth lesion type/severity.
  • Clinical utility: does the classification predict the correct treatment (e.g., distinguishing inflammatory/Type-1–style resorption needing full extraction vs replacement/Type-2–style favoring crown amputation)?
  • Discrimination & granularity: ability to separate clinically distinct lesions without redundant or empty categories.
Decision · Q2
Which system performs better for cats?
▼ Andreasen superior
Finding 2a
Human Andreasen system > AVDC for felines
Mirrors the canine conclusion. Candidate recommendation: adopt Andreasen for cats — but first test Phase 3.
▼ AVDC superior / equivalent
Finding 2b
AVDC ≥ Andreasen for felines
Feline-specific design holds its own. AVDC stays the working standard — still test Phase 3 for a possible improvement.
Converge
All Phase 1 & 2 paths feed the synthesis question below
Regardless of which existing system wins, pool every reliability/validity/utility result and the catalog of lesions that neither system handled cleanly.
3

Can a superior universal feline system be created?

Methodology · Q3
Derive and validate a candidate feline-specific system
  • Gap analysis: identify feline lesions/features captured poorly by both systems (e.g., resorption type 1 vs 2 vs 3, ankylosis/replacement vs inflammatory resorption, alveolar bone expansion, location patterns).
  • System design: build a candidate classification — possibly a hybrid of Andreasen's structure with feline-specific descriptors.
  • Validation: re-score the dataset under the new system; compute reliability, validity, and clinical-utility metrics head-to-head against both AVDC and Andreasen.
  • Generalizability: test on a held-out / independent feline sample to confirm it is "universal," not overfit.
Decision · Q3
Does the new system beat BOTH AVDC and the human (Andreasen) system?
▼ YES — superior on key metrics
Conclusion A
Recommend a new universal feline classification system
More reliable, valid, and clinically useful than AVDC and Andreasen. Propose for adoption + external/multi-center validation.
▼ NO — not superior
Conclusion B
Recommend the best existing system
Adopt whichever won Phase 2 — Andreasen (if 2a) or AVDC (if 2b / Q1 failed). Document why a novel system was not justified.
Node types: premise sets the data foundation · question states the analytic aim · methodology (dashed) lists the metrics that resolve it · decision applies pre-registered thresholds · outcome records the conclusion each path produces. Comparison metrics throughout follow the Peralta (2010) approach — coverage, inter-/intra-observer agreement (κ), validity against ground truth, and clinical-treatment utility.
